Project Overview

At Dtex Systems, we decided to build a V4 version of the application, because the V3 lacked in terms of functionality, features, design and user experience. Because this was such a major upgrade, we decided to build the whole application from the ground up. So we had the chance to incorporate the latest technologies and trends to build it. For the front-end side of the application, we decided to use React as the javascript framework, Redux as the state container and RESTful API's as middleware to insert and read data from Elasticsearch securely.

I have taken up the responsibility of User Interface and User Experience of the entire V4 application. Once I had finished my paper mockups and real-time prototypes with several theme options, after the company's approval of the design, we started building the application and I had developed a large chunk of the application using React and several other technologies and components. Each and every moment at Dtex was so exciting because there was an enormous opportunity to experiment and build.
