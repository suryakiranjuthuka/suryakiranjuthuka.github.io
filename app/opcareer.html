Project Overview

As a consulting and recruiting firm, Optimal Partners needed a system to manage and display both their consultants and open positions. Since we were unable to find an out-of-the-box solution that covered these needs, one needed to be built.

Optimal Partners uses Zoho Recruit as their consultant and position management tool, so this website needed to integrate with that backend. I used the Zoho API to pull information directly from custom fields in the Zoho Recruit database.
