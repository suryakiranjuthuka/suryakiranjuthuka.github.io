Project Overview

This school project was an experimentation in interactive UX design. I wanted to represent my resume in a highly visual, highly interactive way. To build this resume, I used an emerging JavaScript library called D3.js.

This resume shows both an example work resume and an education resume combined into one interactive form. Each element and module is interactive to mouseovers and clicks.
