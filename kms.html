Project Overview

This is a keyword management system (KMS) for use within the company's product, designed and developed by me. This part of the system is where the user manages their keywords within their security system. It’s also where they set their risk levels for different categories and flagged keywords.

This KMS has a backend based in Elastic Search instead of a traditional database. It was developed with PHP as a serverside programming language, which interacts with and brings in data from Elastic Search.
