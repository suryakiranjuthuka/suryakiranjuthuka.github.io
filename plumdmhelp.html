Project Overview

Totally independently designed and developed, this dynamic system is used to keep track of the status of a marketing companyâ€™s client landing page jobs. The system includes a dynamic list, with screenshots, of all possible landing page template designs.

Each salesperson has an account that they use to order new landing pages for their clients. The salesperson logs in, selects a landing page design template, fills out a form with the necessary client information and customization, and submits the request to the web developer. Salespeople can also use their account to view completed or in-progress requests.

The web developer has an admin account which allows them to see all open and completed requests, and change the status of requests (for example, mark a page as completed).
